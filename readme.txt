Facebook plugins, Google Analytics, notifications, website tour. These are just some of the

With the Clearapp for Wordpress plugin, your Clearapp can integrate tightly with your website,
allowing you to deploy all kinds of apps with absolutely no coding.

== Installation ==

1. Sign up for a Clearapp account at http://dashboard.clearapp.net
2. In your Wordpress admin, search for "Clearapp" in the "Admin > Plugins > Add New"
3. Click install
4. Go to "Settings > Clearapp" in your Wordpress admin and click "Install Clearapp"
